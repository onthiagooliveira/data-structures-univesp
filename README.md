# Estudos de Estrutura de Dados e Linguagem C (Univesp)

Repositório destinado ao acompanhamento da disciplina de Estrutura de Dados, com foco no aprendizado aprofundado da linguagem C, manipulação de memória e implementação de algoritmos.

## Sobre a Matéria

A disciplina de **Estrutura de Dados** em C é o alicerce para qualquer desenvolvedor de software que deseja entender como o computador realmente opera. Ao contrário de linguagens de alto nível (como Java), o C exige que o programador gerencie a memória manualmente, utilizando ponteiros e alocação dinâmica.

Os principais pilares deste estudo são:
* **Abstração de Dados:** Uso de `struct` e `typedef` para criar tipos de dados personalizados.
* **Gerenciamento de Memória:** Compreensão da *Stack* (estática) vs *Heap* (dinâmica) e uso de `malloc` e `free`.
* **Estruturas Lineares:** Implementação de Listas Ligadas, Pilhas (Stacks) e Filas (Queues).
* **Algoritmos de Busca e Ordenação:** Análise de eficiência e complexidade.
